# Learnix — Azure Deployment Guide

The deployment guides have been split into two separate documents depending on your preferred approach:

- [**Azure Portal UI Guide**](./deploy/AZURE_DEPLOY_UI.md) — Step-by-step instructions for creating resources manually using the Azure Web Portal.
- [**Azure CLI Guide**](./deploy/AZURE_DEPLOY_CLI.md) — Script-based instructions for creating resources using the Azure CLI (`az`).
